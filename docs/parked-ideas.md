# Parked Ideas

The incubation backlog. Everything here is **deliberately parked**: indexed so
it isn't lost, but **not started** and **not** to be quietly implemented.

An idea leaves this list only when the weekly board promotes it to the single
active implementation lane — see [`weekly-cadence.md`](weekly-cadence.md).

## Promotion

To activate a parked idea:

1. Bring it to the weekly board.
2. Confirm no other implementation lane is active (or explicitly park the
   current one).
3. Move it into the relevant lane in [`lane-index.md`](lane-index.md) and mark
   it active.

Until that happens, parked means parked.
